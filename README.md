# OOP_BRAIN
OOP-Brain, C++'ın yüksek performansını modern OOP prensipleriyle birleştiren modüler bir ANN kütüphanesidir. Polimorfizm, arayüz tabanlı tasarım ve bellek yönetimi odaklı bu proje, karmaşık yapay zeka yapılarını temiz ve genişletilebilir bir mimaride sunar. SOLID ilkeleriyle inşa edilmiş akademik bir mühendislik çalışmasıdır.
